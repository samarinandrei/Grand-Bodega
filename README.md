# Grand Bodega 🍷

Aplicație web simplă pentru meniul unui bar românesc – vezi băuturile, adaugă în comandă, modifică cantități, vezi totalul în lei și salvează comanda chiar dacă închizi tab-ul.

## Live Demo

🌐 **https://grand-bodegamenu-1bgoisz9c-samarinandreis-projects.vercel.app/**  

## Tehnologii folosite

- React + Vite  
- Tailwind CSS v4  
- Zustand (stare globală)  
- react-hot-toast (notificări elegante)  
- localStorage (persistă comanda între refresh-uri)
